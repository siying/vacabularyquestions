# Luca's Vocabulary Mastery Web Application

An interactive, responsive, and beautifully designed study tool created to help Luca master vocabulary words he got wrong or struggled with during his practice exams.

## Features

1. **Study Deck (Tab 1)**: Interactive cards displaying definitions, synonyms, realistic examples, and specific vocabulary pitfalls (e.g. distinguishing *attentive* from *hardworking*, *emulate* from *innovate*, or *provoke* from *scold*). Cards support filtering by "Flat-out Errors" and "Struggled/Corrected" words.
2. **Definitions Quiz (Tab 2)**: Multiple-choice matching quizzes. Can be toggled between "Word to Definition" and "Definition to Word" modes. Features a sleek progress bar and live scoring.
3. **Context Challenge (Tab 3)**: Sentence completion questions based directly on the actual questions from Luca's exam. Upon selecting an answer, it provides detailed, helpful feedback explaining *why* the choice fits, and clarifying any potential points of confusion.

## Technologies Used
- HTML5 (Semantic elements)
- CSS3 (Vanilla CSS, custom properties, dark glassmorphism styling, CSS animations)
- JavaScript (Vanilla ES6 logic, state management, array scrambling)

## Setup and Running Local Dev
Simply open `index.html` in any web browser to run the application locally!
